Pricing

Single source of truth for checkout prices. The checkout and entity-audit-checkout pages fetch these at page load, and /api/checkout/create-session builds a Stripe Checkout Session inline from the Amount value — no pre-existing Stripe Price required. Edit a value here and it takes effect on the next checkout.
